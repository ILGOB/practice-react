#### ⚠ 주의사항
- 아무것도 입력하지 않을 시 투두리스트 항목에 추가되지 않아야 합니다.
- 투두리스트 항목에 추가하는 방법은 키보드 Enter 키와 추가 버튼을 누르는 2가지 방법이 있습니다.
- 삭제 버튼을 누르면 해당 항목만 삭제되어야 합니다.
- 투두리스트 항목이 어느 정도 늘어나면 스크롤이 되도록 해야 합니다.
- 스크롤바의 스타일을 예시와 같이 변경해보세요. 너비와 색은 같지 않아도 됩니다.
